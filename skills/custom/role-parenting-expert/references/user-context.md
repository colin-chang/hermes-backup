# 育儿专家专属用户档案（3岁男童健康基线）

## 基本档案
- **年龄：** 3岁男童（2023年2月28日生于广东深圳，深户）
- **环境变动：** 2026年6月随全家跨国搬迁至加拿大 Hawkesbury, ON
- **医疗身份：** 目前持有加拿大5年期访客签证，落地初期依赖商业医疗保险

## 当前核心护理：额头伤口
- **伤情：** 意外跌倒致额头受伤，创面深度触发缝合指征，已缝针并拆线，处于疤痕干预黄金期
- **当前方案（联合防疤）：**
  - **减张干预：** 免缝胶带 (Steri-Strips/Dressing strips)，通过物理拉力对抗皮肤张力，防止伤口在愈合过程中被拉宽
  - **药物干预：** 芭克 (Kelo-cote) 硅酮凝胶，通过水合作用软化并抑制疤痕组织增生
- **护理重点：**
  - 监控伤口褪红进度（褪红周期是关键指标）
  - 防晒与物理防护（色素沉着 PIH 预防，避免户外活动导致永久性色素沉着）
  - 防止 3 岁儿童因瘙痒抓挠导致二次破溃
- **评估节点：** 每周一次疤痕淡化评估，第 4 周关键评估窗口
- **长期跟踪：** 疤痕 12 个月康复周期（红肿期 → 稳定期 → 干预截止期）

## 跨国健康档案
- **疫苗对齐：** 国内《预防接种证》→ 安省免疫计划(Immunization Connect Ontario)对齐
- **Daycare交叉感染预警：** 入园初期高频呼吸道感染(RSV/流感)和肠胃疾病风险
- **双城医疗策略：** Hawkesbury(安省)居住 + 蒙特利尔(魁省)可能就医的跨省医疗报销分析
- **长途飞行：** 需准备幼儿跨国飞行健康保障包（常备药+护理方案）
