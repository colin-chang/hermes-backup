# 华泰证券 API 与收益率通知能力研究

> 调研日期: 2026-05-29 | 来源: 华泰官网/知乎/叩富网/经济晚报/AI涨乐评测

## 核心结论

华泰证券 **不提供「账户年化收益率达标主动通知」功能**，且这一功能在中国券商行业普遍缺失。

## 华泰现有通知能力（AI涨乐 & 涨乐财富通）

| 功能 | 粒度 | 微信推送 | 说明 |
|------|------|:---:|------|
| 一句话盯盘 | 单只标的 | ✅ | 自然语言设置，支持复杂条件 |
| 止盈止损条件单 | 单只标的 | ❌ | 触发后自动委托 |
| 盯盘套餐 | 单只/多只 | ✅(Clawbot) | 主力动向、日内高低价等 |
| 分批止盈 | 单只标的 | ❌ | 多档自动卖出 |
| 交易复盘 | 账户级别 | ❌ | 收盘后AI生成报告 |
| 早点听 | 账户级别 | ❌ | 盘前8-10分钟播客 |

## 为什么没有账户收益率通知？

1. **技术**: 年化收益率是复合指标，受多持仓盈亏+已实现盈亏+分红+出入金共同影响
2. **合规**: 证监会监管下，「收益率提醒」可能被解读为诱导性承诺
3. **产品**: 出入金会瞬间改变收益率，触发逻辑边界情况过多

## 华泰证券 API 生态

### 三个层级

| 层级 | 平台 | 门槛 | 对象 |
|------|------|------|------|
| QMT（迅投） | 涨乐财富通内嵌 | 10-50万资产 | 个人量化 |
| PTrade（恒生） | 同上 | 10-50万资产 | 个人量化 |
| MQUANT/MATIC | 独立机构平台 | 机构客户 | 私募/自营 |

### API 能力（QMT为例）

- `get_position()` — 持仓查询（市值/成本/浮动盈亏）
- `get_trade_history()` — 历史成交
- `get_asset()` — 账户总资产
- `get_capital_flow()` — 出入金记录

**年化收益率需要自己计算**: `(当前总资产 - 总投入) / 总投入 / 天数 × 365`

### 其他券商量化门槛参考

- 国金证券: 10万 → QMT/PTrade
- TTT证券（国泰海通合并后）: 50万 → QMT/PTrade
- 个人投资者直接API: 大多数券商仅对机构开放
