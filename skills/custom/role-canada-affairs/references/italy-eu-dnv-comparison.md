## 🇮🇹 意大利 Digital Nomad Visa（Nomadi Digitali）

### 核心参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 法律基础 | Inter-ministerial Decree 29 Feb 2024 | 2024年4月5日正式生效 |
| 签证类型 | 双通道：数字游民（自雇） + 远程工作者（雇员） | 两者共享同一框架 |
| 收入门槛 | **~€28,000-30,000/年**（~€2,333-2,500/月） | **欧洲最低之一** |
| 学历要求 | ⚠️ 严格：必须大学学位 + 相关专业资格 | 比西班牙严得多 |
| 工作经验 | 至少 **6 个月**远程工作经验 | |
| 首签 | 1年，入境后 **8天内**必须申请居留卡 | permesso di soggiorno |
| 续签 | 每年续，持续满足条件 | |
| 转 PR | **5 年**合法居留（年满183天） | |
| 入籍 | **10 年** | 需 B1 意大利语 |
| 家属 | ✅ 配偶 + 18岁以下子女（主申获批后可团聚） | 成年子女不适用 |
| 税务优惠 | "新居民"优惠：收入 70-90% 免税（视地区） | 可选方案 |
| 配额 | **无年度配额上限** | |

### ⚠️ 意大利的坑与风险

| 风险 | 详情 |
|------|------|
| 🚧 **政策太新** | 2024年4月才实施，部分领事馆内部流程未完全理顺 |
| 📜 **学历审核极严** | 不同领事馆标准不一，专业资格认证要求比西班牙高一个量级 |
| 🐌 **官僚效率** | permesso di soggiorno 可能等半年，期间无法出入境 |
| 💰 **自雇者要开 VAT** | 自由职业者必须注册 Partita IVA（增值税号），有季度申报合规成本 |
| 🏠 **住房要求前置** | 申请时需提供已签署的租房合同（不接受酒店预订） |

### 三国收入门槛对比（单身）

| 国家 | 月收入要求 | 学历要求 |
|------|:--:|:--:|
| 🇮🇹 意大利 | **~€2,400** ✅ 最低 | ⚠️ 必须有学位+资格 |
| 🇪🇸 西班牙 | €2,849 | 学位或经验二选一 |
| 🇵🇹 葡萄牙 | €3,680 | 无硬性学历 |

### 对 Hawkins 家庭的关键判断

- 意大利对 **妻子（CFA+双硕士+金融背景）** 资格完美匹配
- 但三口之家的家属收入加成标准尚未像西班牙那样明确
- 意大利官僚体系效率是三国中最慢的
- **建议**：再观察1-2年，等意大利 DNV 跑出足够案例后再评估

## 🇬🇷 希腊 Digital Nomad Visa

| 参数 | 值 |
|------|-----|
| 收入门槛 | €3,500/月（单身），家属+20%/人 |
| 首签 | 1-2年 |
| 转 PR | **5 年**（年满183天） |
| 入籍 | PR后再 **7 年**（总共12年，三国中最慢） |
| 税务亮点 | 🔥 境外收入可申请 **7% 统一税率**（需正式转移税务居民身份） |

## 欧盟 DNV → PR 可用国家总结

| 可转 PR | 时间 | 备注 |
|------|:--:|------|
| 🇪🇸 西班牙 | 5年→PR，10年→入籍 | 最成熟+平衡 |
| 🇵🇹 葡萄牙 | 5年→PR，10年→入籍 | 收入门槛最高（三口€6,624），国籍法2026已变 |
| 🇮🇹 意大利 | 5年→PR，10年→入籍 | 收入门槛最低但政策太新 |
| 🇬🇷 希腊 | 5年→PR，12年→入籍 | 入籍最慢，但税务最优（7%统一税） |
| 🇭🇷 克罗地亚 | PR路径**存在争议** | 不建议以PR为目标的申请人 |

> 🇨🇦 对比：加拿大 EE/CEC → PR 1-2年，入籍3年（PR后）。欧洲任何路线的时间线都无法与之竞争。
