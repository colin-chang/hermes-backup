# 跨国家移民政策对比研究方法论

> 适用场景：用户询问"XX 国家移民政策如何？""和加拿大比怎么样？""听说 YY 国放宽了政策"
> 角色仍然是加拿大事务顾问，但需要切换研究目标国家。

## 研究框架（4步法）

### 1. 政策全景抓取
- **官方来源优先**：目标国移民局官网（如 NZ: immigration.govt.nz，AU: immi.homeaffairs.gov.au）
- **权威媒体补充**：移民律所博客 > 移民咨询公司 > 留学中介
- **关键搜索词**：
  - `[国家] immigration policy changes [年份] skilled migrant visa`
  - `[国家] green list / occupation list / shortage list`
  - `[国家] partner work visa spouse conditions wage threshold`

### 2. 核心参数提取
必须提取并记录：
- 技术移民打分体系（如 NZ SMC 6分制、AU 189/190 打分）
- 紧缺职业清单（Green List / MLTSSL / SOL）
- 工资中位数及各类签证的工资门槛
- 配偶工签条件（主申工资门槛、开放程度）
- 审理时间线参考

### 3. 家庭对齐（本文档的核心价值）
- 逐一映射每个家庭成员的职业 → 目标国职业分类（ANZSCO / NOC）
- 逐一映射每个家庭成员的资质 → 目标国打分系统
- 分别模拟"谁做主申"的路径和时间线
- 标注硬门槛（如必须有 job offer、必须本地经验 N 年）

### 4. 对比输出
- 用表格做逐维对比（进度/时间线/就业/生活/护照/税）
- 标注信息源的权威性和时效性
- 明确标注"推测"vs"实锤"

## ⚠️ 西班牙 DNV 速查 — 重要补充：PR ≠ 申根自由迁徙

> 2026.05 调研中发现的普遍误解澄清

### 西班牙两种永居的区别

| 类型 | 西语名称 | 权利范围 |
|------|---------|----------|
| 西班牙国家永居 | *Larga Duración* | **仅限西班牙境内**居住和工作 |
| 欧盟长期居留 | *Larga Duración-UE* | 可向其他EU国申请居留许可，但**需目的国批准**（含收入证明+保险），不能自动自由迁徙 |

- 两种永居均需 **5年合法居住**，UE版本有更严格的收入+保险审查
- 即使拿到欧盟长期居留，搬到阿姆斯特丹/柏林仍需要向目的国重新申请，可能被拒
- **申根区免签旅行（90天/180天）与PR无关** — DNV首年就已享有此权利
- 对比：加拿大PR→全国自由迁徙，无任何附加条件

### DNV 年龄与行业限制澄清

| 问题 | 答案 |
|------|------|
| 最低年龄 | **18 岁** |
| 最高年龄 | **无上限**。60岁也只要收入达标即可申请 |
| 行业/职业限制 | **无。** 只要远程工作、使用计算机通讯手段完成即可。YouTuber/程序员/设计师/在线教师/电商/金融分析师均可 |
| 唯一限制 | ❌ 不可在西班牙本地公司全职上班（自雇者最多20%业务可来自西班牙） |

### DNV 移民监（全程）

| 阶段 | 居住要求 | 违规后果 |
|------|------|------|
| DNV 期间（第1-5年） | 每年≥**183天** + 单次离境≤**6个月** + 5年累计离境≤**10个月** | 续签被拒→居留中断→5年时钟重置 |
| 获得 PR 后 | 单次离境≤**12个月** | PR 被撤销 |

### 续签风险（最被低估的坑）

每次续签（1→2年→2年）都需要**重新审查**：收入+工作关系+保险+居住天数。
YouTube 收入如果某年大幅下滑，续签可能被拒→时钟归零。
不是"5年到了自动给PR"，而是"每次续签都像重新申请一次"。
