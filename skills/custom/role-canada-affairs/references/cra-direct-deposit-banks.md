# 加拿大各银行 CRA Direct Deposit 设置路径

> 来源：各银行官网及 CRA 官方页面（2026.05 确认）
> 更新时机：发现新银行支持或现有银行路径变更时更新

## CIBC（帝国商业银行）✅ 已验证

**设置路径：**
- 📱 **手机App**：登录 → 菜单 → Customer Services → Set up direct deposit with Canada Revenue Agency
- 💻 **网银**：登录 cibc.com → 左侧导航 → Customer Services → Set up direct deposit with Canada Revenue Agency
- 🔄 **备选路径**（上面找不到时）：Request Centre → More service requests → Register for CRA direct deposit

**所需材料：** SIN 号码、选择接收退款的账户（Chequing 或 Savings）
**处理时间：** 下一个工作日
**客服电话：** 1-800-465-2422

**关键提示：** CIBC 是五大行中 App 体验最好的之一，该功能100%支持。不需要 void cheque、不需要报税记录、不需要 CRA 账号——银行直接传输给 CRA。

## 通用方法（各银行适用）

1. 登录银行 App 或网银
2. 搜索框搜 "CRA" 或 "Direct Deposit"
3. 如无搜索结果，在菜单中找 "Customer Services" 或 "Account Services"
4. 实在找不到 → 打银行客服，说 "I want to set up CRA direct deposit through my online banking"
5. 若该银行不支持线上绑定 → 改用邮寄表单方案（见 role-memory.md）
