# Mattermost 频道描述撰写规范

## 触发条件

当需要为 Colin 的 Mattermost 频道撰写「频道标题」中的描述文字时，遵循此规范。

## 现有频道描述参考

| 频道 | 定位词 | 风格 |
|------|--------|------|
| 媒体运营 | 「创意孵化实验室」 | 打造优质内容的... |
| 大话财经 | 「市场分析窗口」 | 洞察财富增长的... |
| 移民移居 | 「实战路线指南」 | 移居规划的... |
| 育儿百科 | 「循证知识库」 | 守护孩子成长的... |

## 结构公式

```
[动词+领域] 的 [「引号定位词」]。
→ 展开覆盖领域（列举具体主题，用分号或破折号串联）
→ 收束：受众价值声明（不贩卖焦虑/只讲干货/助你...）
```

## 约束

- **字数**：≈100 字，精炼不啰嗦
- **内容**：描述频道覆盖的主题领域，**不是**描述 AI 角色的身份和能力
- **语气**：务实、不浮夸、不制造焦虑
- **格式**：先查询现有频道描述（`docker exec mm-postgres psql ...`）对齐风格
